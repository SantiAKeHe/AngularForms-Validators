Kendo UI for Angular

Kendo UI components integrate natively with Angular Reactive Forms via ControlValueAccessor — use formControlName directly, no wrappers needed.

ComboBox — pick from a list or type a custom value

<kendo-combobox> lets the user select from a predefined list or type a custom value that is not in the list.

DropDownList — pick only from a predefined list

<kendo-dropdownlist> restricts the user to the provided options — typing a custom value is not allowed.
