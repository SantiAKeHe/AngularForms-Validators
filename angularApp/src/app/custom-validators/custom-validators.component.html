Topic 03

Custom Validators

Learn how to build the three types of custom validators in Angular 19: simple validators, factory validators (with parameters), and cross-field validators.

Simple Validator — noSpacesValidator

Returns noSpaces: true if the value contains any spaces. Applied directly on a single FormControl.

Factory Validator — minAgeValidator(18)

A validator factory that accepts a parameter. Returns minAge: { required, actual } when the value is below the minimum.

Live State

Value:   {{ minAgeForm.value | json }}
Minimum: 18
Actual:  {{ age.value }}
Status:  {{ minAgeForm.status }}

Cross-field Validator — passwordMatchValidator

Applied at the FormGroup level. Returns passwordMismatch: true when the two fields do not match.

@if (passwordForm.hasError('passwordMismatch') && confirmPassword.touched) {
Passwords do not match.
}

Status:  {{ passwordForm.status }}
